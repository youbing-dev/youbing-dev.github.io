## 项目作品

以下是我主导或深度参与的部分项目，涵盖分布式系统、API 网关和监控告警方向。

---

### 分布式任务调度平台

> 基于 Spring Boot + Quartz 构建的分布式任务调度系统，支持动态调度、失败重试和执行日志可视化。

**技术栈：** Java, Quartz, MySQL

- [GitHub 仓库](https://github.com/youbing-dev/task-scheduler)
- [在线演示](https://scheduler.youbing.dev)

---

### API 网关 & 统一鉴权服务

> 自研轻量级 API 网关，支持 JWT 鉴权、请求限流、路由转发和灰度发布，日均处理 500 万次请求。

**技术栈：** Spring Cloud, Redis, Netty

- [GitHub 仓库](https://github.com/youbing-dev/api-gateway)

---

### 实时监控告警系统

> 基于 Prometheus + Grafana 定制的监控告警平台，支持自定义指标采集、多通道告警和异常自动诊断。

**技术栈：** Prometheus, Grafana, Python

- [GitHub 仓库](https://github.com/youbing-dev/monitor-alert)
- [在线演示](https://monitor.youbing.dev)
